# InkMerge - Custom Shopify Theme

InkMerge DTF Transfer Business - Custom Shopify Theme with Lead Magnet Funnel & SEO Optimization

## Project Overview

Complete DTF (Direct-to-Film) transfer business website featuring:

- ✅ **Custom Shopify Theme** with gang sheet builder integration
- ✅ **Lead Magnet Funnel** with Acumbamail email automation
- ✅ **SEO Optimization** with comprehensive schema markup
- ✅ **IndexNow Integration** for instant search engine indexing
- ✅ **DTF Application Guide** (50+ page comprehensive resource)

## Features

### Shopify Theme
- Gang sheet builder product template
- SEO-optimized blog with 99 unique articles
- Mobile-responsive design
- Schema.org structured data
- Visual breadcrumbs
- Testimonials and reviews section

### Lead Magnet System
- Landing page hosted on Replit
- Python backend API
- Acumbamail integration
- Automated welcome emails with GUIDE10 discount code
- Thank-you page with guide download

### Technical Stack
- **Backend**: Python 3.11
- **Frontend**: Liquid (Shopify), HTML5, CSS3
- **Email Marketing**: Acumbamail API
- **Search Integration**: IndexNow API
- **Deployment**: Replit for funnel, Shopify for theme
